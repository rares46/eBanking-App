# eBanking-App
Application containing different usefull implementation commands in simulating a real eBanking App.
Users can create an account, they can add money, transfer to other users, exchange money (reading from a .csv file) buy stocks or crypto. There is a premium option that gives the user advantages. Also, the application recommends stocks.
The application uses Singleton and Builder design patterns. Each command is self-explanatory and easy to read and understand.
